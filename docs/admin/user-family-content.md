# 用户家庭与互动内容规则

## 用户管理

- 支持用户查询、状态管理、基础信息查看。
- 不在后台展示敏感凭证。
- 禁用用户前必须确认影响范围。

## 家庭管理

- 支持查看家庭基础信息、成员、共享状态。
- 成员管理必须区分管理员和成员。
- 删除成员、退出家庭、解散家庭必须有确认。

## 菜篮子 / 采购

- 支持按家庭或用户查看采购清单。
- 采购记录按日期归档。
- 价格记录要能关联食材。

## 收藏与浏览

- 支持按用户查看收藏和浏览历史。
- C 端取消收藏后后台数据应同步。

## 评论 / 晒菜

- 当前不是 MVP 主线。
- 后续如启用，必须接入审核状态。
