# 首页运营规则

## 覆盖模块

- 顶部导航。
- Banner。
- 推荐位。
- 时令食材。
- 菜系。
- 场景菜单。

## 顶部导航

- 后台可配置导航名称、类型、排序、状态。
- C 端首页按后台启用状态和排序展示。
- 默认导航必须有兜底内容。

## Banner / 轮播图

- 支持图片、标题、描述、跳转、排序、上下架。
- 图片必须通过统一上传资源。
- C 端点击跳转必须可验证。

## 推荐位

- 可关联菜谱、食材、菜单或运营资源。
- 必须支持排序、上下架。
- C 端只展示已启用内容。

## 场景菜单

- 可绑定多道菜谱。
- 面向家常、快手、汤类、早餐、减脂等入口。

## 验证

后台修改后必须验证 C 端首页同步展示。
