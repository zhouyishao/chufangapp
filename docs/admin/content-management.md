# 内容管理规则

## 覆盖模块

- 菜谱管理。
- 食材管理。
- 水果管理。
- 调料管理。
- 酒水管理。

## 菜谱管理

核心能力：

- 新增、编辑、详情、删除。
- 上架、下架、推荐、审核。
- 维护封面图、基础信息、用料、步骤、标签、分类。

关键字段见：`docs/backend/field-models.md`。

## 食材 / 水果 / 调料 / 酒水

核心能力：

- 新增、编辑、详情、删除。
- 发布、隐藏、推荐。
- 维护封面图、分类、标签、单位、价格、时令信息。

## 页面规则

- 列表字段不可过多，复杂信息进入详情页。
- 新增 / 编辑页按基础信息、图片、分类标签、展示配置分区。
- 和 C 端展示相关字段必须在 API 中返回。

## 联动验证

修改内容后必须验证：

- C 端列表是否展示。
- C 端详情是否展示。
- 图片是否正常加载。
- 下架后 C 端是否不可见。
