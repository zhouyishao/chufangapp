# 配色系统

## C 端主色

| 用途 | 色值 | 说明 |
|---|---|---|
| 主色 | `#7A8B6F` | 鼠尾草绿，按钮、选中态、强调 |
| 背景 | `#F5F1EA` | 暖奶油色，页面底色 |
| 卡片 | `#FFFDFC` | 暖白，卡片和列表项 |
| 主文字 | `#2F2F2F` | 标题和正文 |
| 辅助灰 | `#B7AEA1` | 次要文字、分割线 |
| 点缀橙 | `#C27B48` | 价格、重要标签、收藏，少量使用 |

## 后台色彩

- 以 Ant Design 主题为基础。
- 不额外引入另一套后台视觉体系。
- 状态色使用 Ant Design 默认语义色。

## 使用规则

- 禁止纯黑 `#000000` 作为大面积背景或文字。
- 禁止页面同时出现多套品牌色。
- 价格、警告、删除等语义色必须克制使用。
