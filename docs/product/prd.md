# 产品 PRD

## 产品定位

家庭时令菜谱 App（家里有菜）面向家庭日常做饭场景，帮助用户围绕时令食材、菜谱、菜篮子和家庭共享完成买菜与做饭闭环。

## MVP 核心目标

- C 端展示后台配置的首页内容、菜谱、食材、时令食材。
- 用户可查看菜谱详情、食材详情、加入菜篮子、收藏、浏览历史。
- 后台可管理菜谱、食材、分类、推荐、Banner、用户和内容。
- 后端使用统一 `server/` 服务真实落库，并向后台和 C 端提供接口。

## 核心用户路径

1. 用户打开首页，查看推荐菜谱和时令食材。
2. 用户进入菜谱详情，查看用料和步骤。
3. 用户将菜谱或食材加入菜篮子。
4. 用户完成采购后记录采购和价格信息。
5. 用户在我的页面查看收藏、浏览、采购记录、家庭信息。

## 当前功能范围

- 首页运营内容展示。
- 菜谱列表、菜谱详情、我的菜谱。
- 食材列表、食材详情、价格记录。
- 菜篮子、采购记录。
- 用户登录、个人信息、家庭管理。
- 后台内容管理与 C 端展示联动。

## 暂不作为主线的功能

- 社区发帖和复杂评论体系。
- 商城、支付、订单交易。
- AI 推荐。
- 排行榜。
- 复杂角色权限产品化。
