# 核心字段模型

本文件承接旧后台 PRD 和 E2E 任务书中的字段说明，避免字段散落在多个长文档。

## Banner

- 标题。
- 描述。
- 图片 URL。
- 跳转类型。
- 跳转目标。
- 排序。
- 状态。

## 菜谱

- 标题。
- 描述。
- 封面图。
- 分类。
- 标签。
- 难度。
- 耗时。
- 口味。
- 热量。
- 用料。
- 步骤。
- 状态。
- 推荐状态。

## 食材 / 水果

- 名称。
- 描述。
- 封面图。
- 分类。
- 标签。
- 时令月份。
- 价格。
- 单位。
- 保存方式。
- 挑选建议。
- 状态。

## 调料 / 酒水

- 名称。
- 描述。
- 封面图。
- 分类。
- 单位。
- 适用场景。
- 状态。

## 价格历史

- 食材 ID。
- 记录日期。
- 总价。
- 购买量。
- 单位。
- 换算后单价。
- 备注。

## 用户、家庭、采购

- 用户：昵称、手机号、头像、状态。
- 家庭：名称、成员、管理员、共享状态。
- 采购：家庭 ID、菜谱来源、食材项、已采购状态、完成时间。

## 配置表

- 配置 key。
- 配置值。
- 配置类型。
- 状态。
- 说明。
