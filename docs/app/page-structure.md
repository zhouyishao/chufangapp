# C 端页面结构

## 核心页面

- 首页。
- 食材列表。
- 食材详情。
- 菜谱列表。
- 菜谱详情。
- 菜篮子。
- 我的。

## 用户资产页面

- 收藏。
- 浏览历史。
- 我的菜谱。
- 创建菜谱。
- 采购历史。
- 个人资料。
- 家庭管理。

## 登录注册页面

- 登录。
- 手机登录。
- 注册。
- 忘记密码。

## 页面规则

- C 端页面必须使用真实接口数据或明确的本地演示数据，不允许长期假数据污染。
- 所有图片通过统一资源处理函数。
- 每个页面必须有 Loading、Empty、Error。
- 页面入口和路由必须同步更新 `docs/app/route-map.md`。
